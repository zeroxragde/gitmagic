# gitmagic
App para ayudarte con tus tiempos en GitLab  
-Configuracion-  
Con una interfaz sencilla  
![image](https://user-images.githubusercontent.com/4545852/213816691-48baa8ad-63f2-416b-9f7d-a9d4761df2c8.png)  
GitMagic te permite estar al pendiente de tus ISSUES  

![image](https://user-images.githubusercontent.com/4545852/213816876-e0280652-aa9e-400d-af41-ddec0f49592a.png)  
Solo debes ingresar tu token y usuario de GitLab
- En GitLab ingresa a User Settings -> Access Tokens 
- Y crea uno con todos los permisos.
- Puedes ingresar una hora del dia y GitMagic te pedira que ingreses tus spends abriendo la interfaz de Spends:

![image](https://user-images.githubusercontent.com/4545852/213817326-cf0fb4f5-10c1-4f44-82cf-421c0e7b33cf.png)  

La interfaz te pedira el issue y lo cargara en el webview donde podras confirmar, ya solio debes ingresar las horas o minutos que invertiste 
y dar clic en agregar  

![image](https://user-images.githubusercontent.com/4545852/213817473-4c447ce2-6420-4c67-8f06-08a1dbfe4803.png)  
El reporte mensual te carga todos los Issue abiertos o cerrados y los merges que has autorizado y te contabiliza tus spends
